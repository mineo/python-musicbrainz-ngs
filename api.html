API¶

General¶

musicbrainzngs.auth(u, p)¶

Set the username and password to be used in subsequent queries to the MusicBrainz XML API that require authentication.

musicbrainzngs.set_rate_limit(new_interval=1.0, new_requests=1)¶

Sets the rate limiting behavior of the module. Must be invoked before the first Web service call. Specify the number of requests (new_requests) that may be made per given interval (new_interval).
